This is the release version of the Aurora Borealis project
Run the executive located in the Release folder (May require additional installation of C++ Redistributable from Microsoft)

Controls for program:

Mouse control: Drag to move camera position, Scroll to zoom in or out
Keyboard control:
Up: Tilt camera up, look down
Down: Tilt camera down, look up
Right: Rotates camera clockwise, looks right
Left: Rotates camera counter-clockwise, looks left
F: Toggle full screen mode
W: Shows wireframe of objects
X: Exits program
